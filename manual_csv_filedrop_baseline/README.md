# manual_csv_filedrop_baseline

Ez a mappa a manual CSV / file-drop forrás **tiszta baseline példánya**.

A tesztfuttatók innen tudják visszaállítani a `manual_csv_filedrop` working input mappát, ha egy teszteset közben a working mappa tartalma módosult.

Normál esetben ezt a baseline mappát nem módosítjuk. A célja az, hogy a manual CSV tesztek reprodukálhatóan újraindíthatók legyenek ugyanabból a kiinduló állapotból.
